# Claude へのセキュリティルール

## 🔒 絶対に読み取り・出力しないファイル
- `.env` / `.env.*`（APIキー・パスワードが入っている）
- `*.pem` / `*.key`（秘密鍵）
- `.aws/` / `.ssh/`（クラウド・SSH認証情報）

## 🚫 禁止コマンド
- `printenv` / `env`（環境変数の出力）
- `cat .env*`（環境変数ファイルの表示）
- `curl` / `wget`（外部への通信）
- `rm -rf`（ファイルの一括削除）
- `git push --force`（履歴の強制書き換え）
- `sudo`（管理者権限での実行）

## ✅ コーディングルール
- APIキーやパスワードをコードに直接書かない
- 外部入力は必ずバリデーションする
- エラーメッセージに内部情報を含めない

## 📝 運用ルール
- 機密ファイルの内容を会話に貼らない
- 構造だけ聞く（例：「.envにはどんなキーが必要？」）
- ログを貼る前に機密情報をマスク（隠す）する
